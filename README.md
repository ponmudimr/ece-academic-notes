# ECE Academic Notes

A collection of lecture materials, PDFs, PPTs, and question banks for ECE Semester 1.

## Subjects Covered

- **BEEE** - Basics of Electronics Engineering
- **Chemistry** - Engineering Chemistry (with Lab Records)
- **CPS** - Communication and Problem Solving
- **Maths** - Engineering Mathematics II
- **Physics** - ElectroMagnetism and Modern Physics

---

## How to Download

### Linux

**Step 1 — Install Git:**

```bash
sudo apt update
sudo apt install git
```

**Step 2 — Clone the repository:**

```bash
git clone https://github.com/ponmudimr/ece-academic-notes.git
```

**Step 3 — Navigate into the folder:**

```bash
cd ece-academic-notes
ls "SEM 1/"
```

---

### Windows

**Step 1 — Install Git:**

- Download Git from https://git-scm.com/download/win
- Run the installer and follow the setup wizard (keep default options)

**Step 2 — Open Git Bash:**

- Right-click on your Desktop or any folder
- Select **"Git Bash Here"**

**Step 3 — Clone the repository:**

```bash
git clone https://github.com/ponmudimr/ece-academic-notes.git
```

**Step 4 — Navigate into the folder:**

```bash
cd ece-academic-notes
```

The folder will appear in File Explorer where you ran Git Bash.

---

### macOS

**Step 1 — Install Git:**

Open Terminal and run:

```bash
xcode-select --install
```

Or install via Homebrew:

```bash
brew install git
```

**Step 2 — Clone the repository:**

```bash
git clone https://github.com/ponmudimr/ece-academic-notes.git
```

**Step 3 — Navigate into the folder:**

```bash
cd ece-academic-notes
ls "SEM 1/"
```
